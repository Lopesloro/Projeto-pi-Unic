📖 Sobre o projeto

Sobre o PC Builder AI

O PC Builder AI nasceu como Projeto Integrador com o objetivo de democratizar o acesso à montagem de computadores no Brasil.

Muitas pessoas têm dificuldade em escolher peças compatíveis dentro do orçamento — a IA elimina essa barreira, entregando uma configuração tecnicamente sólida em segundos.

Desenvolvido com HTML, CSS, JavaScript vanilla e integração com a API Gemini do Google, o projeto une tecnologia acessível com inteligência artificial de ponta.
